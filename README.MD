# ESTUDOS SOBRE REPETIÇÃO

Nesta aula estudamos sobre o uso das estruturas de código "*For i in range(1, 11)*" e "*while*"
para melhorar o programa em quesitos de repetição!!

o For serve para repetições em que sabemos o número total de "tentativas"
o While serve para inúmeras tentativas, ele se repete até o usuário digitar a exceção correta.

O programa foi executado com 10 candidatos, para alterar para 50 devemos mudar:
*for i in range(1, 11)*
Para: 
*for i in range(1, 51)*
